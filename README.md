# Flying-blue-app
Flying Blue App is a web/mobile application concept designed to enhance the experience of users within the Flying Blue loyalty ecosystem. The app focuses on simplifying how users track, manage, and maximize their miles, while providing a more intuitive and modern interface compared to existing solutions.

The project explores features such as real-time miles tracking, partner integrations, and personalized recommendations to help users get the most value from their rewards. Similar to existing Flying Blue tools, users can earn and monitor miles through purchases, bookings, and partner services, all in one place.

This repository serves as a development and design prototype, showcasing both the technical implementation and user experience improvements for a more seamless loyalty program interaction. It is intended for learning, experimentation, and demonstrating how airline reward systems can be modernized through better UX and integration.
